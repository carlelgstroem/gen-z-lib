
# GenZ-Lib


### Usage
```c++
git clone https://github.com/yourusername/gen-z-lib.git
```
```c++
cd gen-z-lib
```

Create a Build Directory
```
mkdir build
```
```
cd build
```
Configure the Project with CMake

```
cmake ..
```
Build the Library

```
cmake --build .
```
Install the Library

```
sudo cmake --install . --prefix /usr/local/
```
